Placeholder

Place holder Caret Animation

Placeholdem is a JavaScript plugin that animates place holder carets on inputs and textareas. The placeholder value will incrementally delete on focus, and restore on blur.

Try It

Get It

Use It

<!-- add placeholder to input or textarea --/>
<input name="fieldname" placeholder="Placeholder Value" />
// run Placeholdem on all elements with placeholders
Placeholdem( document.querySelectorAll( '[placeholder]' ) );
